# Project-Showcase-Gallery
Project Showcase Gallery is a responsive web application designed to display and organize final-year student projects in an attractive and user-friendly gallery interface. Built using HTML, CSS, JavaScript, and Flask, this platform allows users to browse project details, view individual project pages, and navigate through a visually appealing collection of academic work.
